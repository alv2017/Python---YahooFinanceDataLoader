# How to run tests ?

Project Structure:
```
YahooFinanceDataLoader_ (project root directory)
						|_examples
						|_tests
						|_YahooFinanceDataLoader
```
						
The tests can be run from the project root directory by issuing the command:
```bash
$ python -m unittest
```

If you want a more verbose output, try running

```bash
$ python -m unittest -v
```
